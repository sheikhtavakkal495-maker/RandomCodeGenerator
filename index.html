How It Works

Frontend → Python Flask API → cryptographic generation → response.

01
Choose Code Type
Pick from 8 types. Each maps to a dedicated Python generator using secrets.choice(), secrets.token_hex(), or uuid.uuid4().
02
Configure Parameters
Set length (4–128), quantity (1–100), charset options, and an optional prefix. Flask validates everything server-side.
03
Python Generates & Scores
Each code is generated with OS entropy and scored using Shannon's formula: len × log₂(charset_size).
04
Copy or Export
Copy individual codes, copy all, or download as a .txt file. Every code logged to your session history.
